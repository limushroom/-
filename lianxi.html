图片下端渐隐模糊效果

这种效果使用CSS的线性渐变和背景模糊滤镜实现。通过调整下方控制条，您可以自定义模糊的高度和强度。

实现代码

// HTML结构
<div class="image-container">
    <img src="image.jpg" alt="Example">
    <div class="fade-blur-bottom"></div>
</div>

// CSS样式
.image-container {
    position: relative;
    width: 100%;
    height: 500px;
    overflow: hidden;
}

.fade-blur-bottom {
    position: absolute;
    bottom: 0;
    left: 0;
    width: 100%;
    height: 40%; // 可调整模糊区域高度
    background: linear-gradient(to top, 
               rgba(255, 255, 255, 1) 0%, 
               rgba(255, 255, 255, 0.8) 20%,
               rgba(255, 255, 255, 0.4) 40%,
               rgba(255, 255, 255, 0.2) 60%,
               rgba(255, 255, 255, 0) 100%);
    backdrop-filter: blur(5px); // 可调整模糊强度
    pointer-events: none;
}
